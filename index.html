JavaScript

Contexto histórico

A linguagem de programação JavaScript surgiu em meados da década de 1990, sendo desenvolvida por Brendan Eich na Netscape Communications Corporation. Na época, a web estava ganhando popularidade rapidamente, e havia uma necessidade crescente por tecnologias que permitissem interatividade e dinamismo nas páginas da web.

JavaScript foi concebido como uma linguagem de script para ser executada no lado do cliente, ou seja, no navegador web do usuário. Sua função inicial era fornecer recursos para manipulação e interação com elementos HTML e CSS, tornando possível criar páginas web mais dinâmicas e interativas.

Linguagem

JavaScript é uma linguagem de programação de alto nível, dinâmica e orientada a objetos. Sua sintaxe é inspirada em várias linguagens, incluindo C e Java, tornando-a relativamente fácil de aprender para quem já tem experiência em programação.

Agora vamos abordar alguns aspectos básicos da linguagem e como a linguagem funciona:

Variáveis

Existem três forma de declarar variáveis em JavaScript usando as palavras-chave ´var´, ´let´ ou ´const´. Ela é uma linguagem dinâmica, o que significa de que você não precisa especificar o tipo de dado que a variável irá receber.

var nome = "João";
let idade = 30;
const peso = 70.5;

Agora as diferenças...

  • var: É a forma padrão de declarar variáveis.
  • let: O valor da variável pode ser alterado ao longo do programa.
  • const: A variável tem o valor constante e naõ pode ser mudado.

Deixando o ´var´ um pouco de lado agora, devemos utilizar o ´const´ o máximo de vezes possível, para garantir a confiabilidade do código, isso falando profissionalmente.

    Estruturas do Switch

    O ´switch´ é uma estrutura de controle de fluxo em JavaScript que permite testar múltiplos casos e executar diferentes blocos de código com base no valor de uma expressão. O ´switch´ é útil quando você tem uma expressão que pode ter vários valores e deseja executar diferentes ações com base nesses valores.

    const fruta = "maçã";

    switch(fruta){
    case "banana":

    console.log("banana é a fruta");
    break;
    case "maçã":

    console.log("maçã é a fruta"); break;

    default:

    console.log("Fruta não encontrada!");
    }
  • O ´break´ para a execução.
  • O ´default´ É executado se nenhum dos casos corresponder à expressão.

    • Laços de repetição

    Estutura do for

    O loop ´for´ é usado quando você sabe exatamente quantas vezes deseja executar o bloco de código. Ele consiste em três partes: inicialização, condição de continuação e expressão final.

    for(inicialização; condição; expressão final){
    // Código a ser executado repetidamente
    }

    Estutura do while

    O loop ´while´ executa um bloco de código enquanto uma condição especificada for verdadeira. A condição é verificada antes de cada iteração.

    while(condição){
    // Código a ser executado repetidamente
    }

    Estrutura do while

    O ´loop do...while´ é semelhante ao while, mas a condição é verificada após a execução do bloco de código, o que garante que o bloco de código seja executado pelo menos uma vez, mesmo se a condição inicialmente for falsa.

    do{
    // Código a ser executado repetidamente
    }while(condição);

    Arrow function

    É básicamente uma função com uma sintaxe mais resumida.

    Estrutura da arrow função

    1. Nescessário que crie uma variável constante;
    2. O nome da variável é o nome da função;
    3. O ponteiro ´=>´ vai apontar para o que a função vai realizar;
    4. A evocação vai ser da mesma maneira que a função comum.
    const testeArrow = () => console.log("Isso também é uma função.");

    testeArrow();

    Array e listas

    Isso é uma lista:

    const numeros = [1,2,3,4]

    Acessando um índice da lista:

    console.log(num [1]);
    console.log(num [2]);
    console.log(num [3]);

    Colocando itens dentro da lista usando o ´push()´.

    num.push(5);
    num.push(6);

    console.log(num)

    Excluíndo itens dentro da lista usando o ´pop()´.

    num.pop(5);

    console.log(num)

    Preenche todos os elementos da lista com o número desejado ´fill()´.

    num.fill(1);

    console.log(num)

    Remove o primeiro elemento do lista ´shift()´.

    num.shift();

    console.log(num)

    Adiciona um elemento no início da lista ´unshift()´.

    num.unshift(7);

    console.log(num)

    Inverte a ordem dos elementos da lista ´reverse()´.

    num.reverse();

    console.log(num)
